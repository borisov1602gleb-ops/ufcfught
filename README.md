# Персональный помощник юриста

Браузерное веб-приложение для юристов: вход → хаб навыков → работа с навыком.
Платформа модульная — навыки, практики, документы и роли описываются **данными**,
а не захардкожены в ядре.

## Архитектура

```
Вход → Авторизация → Хаб навыков → Выбор навыка → Работа с навыком
```

- **backend/** — FastAPI. Аутентификация (JWT), ролевая модель (RBAC), реестр навыков.
  Доступ задаётся **правами на действия** (permissions), а не списком навыков.
- **frontend/** — React + Vite + TypeScript. Окно авторизации, хаб навыков.

### Ключевые точки расширения (всё — данные, ядро не трогаем)
| Что добавить | Где |
| --- | --- |
| Право | `backend/app/rbac/permissions.py` |
| Роль | `backend/app/rbac/roles.py` |
| Навык (манифест) | `backend/app/skills/registry.py` |
| Пользователь | `backend/app/users/store.py` (позже — БД) |

## Этап 1 (готово)
Окно авторизации + бэкенд аутентификации с ролевой моделью и хабом-заглушкой.

Демо-доступы: `lawyer / lawyer123` (юрист), `admin / admin123` (администратор/методолог).

## Запуск

### Бэкенд
```bash
cd backend
python3 -m venv .venv && . .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env   # при необходимости отредактировать
uvicorn app.main:app --reload --port 8000
```

### Фронтенд
```bash
cd frontend
npm install
npm run dev   # http://localhost:5173 (запросы /api проксируются на :8000)
```
